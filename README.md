# 🩺 Diabetes Prediction using Machine Learning

## 📌 Overview

This project uses Machine Learning to predict whether a person has diabetes based on medical attributes such as glucose level, BMI, blood pressure, insulin level, and age.

The model is built using Logistic Regression and follows a complete machine learning workflow including data preprocessing, feature scaling, visualization, model training, evaluation, and prediction.

## 🚀 Features

* Data Cleaning and Preprocessing
* Data Visualization
* Feature Scaling using StandardScaler
* Logistic Regression Model
* Model Evaluation using Accuracy Score
* User Input-Based Prediction System

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn

## 📊 Dataset Features

* Pregnancies
* Glucose
* Blood Pressure
* Skin Thickness
* Insulin
* BMI
* Diabetes Pedigree Function
* Age
* Outcome (Diabetic / Non-Diabetic)

## 🤖 Machine Learning Algorithm

**Logistic Regression** was used to train the model and predict diabetes based on patient health data.

## 📈 Project Workflow

1. Import Dataset
2. Data Cleaning
3. Data Visualization
4. Feature Scaling
5. Train-Test Split
6. Model Training
7. Model Evaluation
8. Prediction

## 🎯 Results

The model was successfully trained and tested, achieving good accuracy in predicting diabetes using patient medical information.
